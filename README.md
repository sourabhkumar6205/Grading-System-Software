# Grading System Software (HTML + JavaScript)

This project is a simple grading system tool built with basic HTML, CSS, and JavaScript.  
It allows users to enter marks and instantly displays the grade along with a motivational message.

### Features
- Enter marks and check grade instantly
- Displays personalized message based on performance
- Pass or Fail color indication (green for pass, red for fail)
- Animated gradient background for a modern UI
- Completely client-side and lightweight

### Grade Criteria
| Marks Range | Grade     | Message                    |
|------------|-----------|----------------------------|
| 91 - 100   | A++       | Excellent Performance      |
| 81 - 90    | A         | Very Good Performance      |
| 71 - 80    | B+        | Good Performance           |
| 61 - 70    | B         | Keep it up                 |
| 51 - 60    | C         | Pass Ho Gye                |
| 0 - 50     | Fail      | Phir Se Exam Do            |

### How It Works
1. User enters marks in the input box
2. Clicks the **Check** button
3. The script calculates grade and shows the result on screen
4. Result color changes based on Pass or Fail status

### Technologies Used
- HTML
- CSS (inline modern styling)
- JavaScript (basic logic and DOM manipulation)

### How to Run
- Download or clone the repository
- Open the HTML file in any web browser
- Enter marks and check your grade
